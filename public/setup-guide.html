🎯 Step 1: Create a Twitch Bot Account

If you haven't already, create a separate Twitch account to act as your bot. You can do this at twitch.tv/signup. Give it a name like MyChannelBot so it's easy to recognize.

🔑 Step 2: Generate Your OAuth Token

The TMI Token Generator has been discontinued, but you can still generate a valid OAuth token using this trusted tool:

Get OAuth Token at twitchtokengenerator.com

Use the "Chat Token" option, and make sure you're logged in as your bot account when generating the token. You’ll receive something like oauth:abc123xyz456. Copy this for the config screen.

📺 Step 3: Enter Your Channel Name

This is your main Twitch channel (not the bot). For example, if your channel is twitch.tv/superstreamer, enter superstreamer — no # or special symbols needed.

✅ Step 4: Save and Launch

Once you've filled in your bot username, token, and channel name, hit Save Config and return to the dashboard to Launch Bot. Your Twitch welcome bot will be live!

🛡️ A Note on Security

Third-party token generators are convenient but come with trust risks. If you're a developer, it's best to register your own Twitch app and generate tokens directly via the official OAuth flow. See the Twitch OAuth documentation for more info.

❓ Still Need Help?

If you're stuck, try logging out and back into Twitch, double-checking your token format, or asking in a developer Discord or forum. Make sure your bot account has sent at least one message in the channel chat!
